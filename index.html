King Wheels Slot Mania

King Wheels Slot Mania was a final project I created for my intro Mobile Apps class built entirely in Android Studio. The game simulates a slot machine game, but with a disabled twist to reflect my disability by having the images be mostly different versions of lavish wheelchairs. There are some special features of this game, when launched you can set your name, which will be remembered in the devices' SQLite storage. You can also take a photo and set it to your profile. Once the game is completed the game will congratulate you and then display your current GPS location.

The biggest challenges I faced while getting everything to sync up by having the "slots" images finish spinning one after another consistently. As of now I have not updated the game, but I might in the future to have it work with iOS devices better.

Photography Website

BJM Photography website was a final group project between me and two other students for our 2nd year Web Dev class. The project was split up into two sections a front end that was written in Vue.JS and Javascript and a backend database that was written in TypeORM. The goal of the website is to allow a user to select a location from an integrated Google Maps page with pins on it, then book an appointment with a form and a calendar date picker. We included a portfolio section that could show off the photographers own work or get an idea of what a photo shoot in one of the Google Map locations would look like. Also, if the user wanted to create an account and log into the website and view their account information on another page. There is also a staff page where staff of the website can edit information or update changes.

The biggest challenged we faced was working with Google's API for Google Maps integration because the API only allows certain features and limited usage unless you pay thousands for a license. This made development hard, and we got 99% functionality working. We ultimately wanted to have the ability to add locations dynamically instead of hard coding them. Nonetheless, there is still a proof of concept when visiting the webpage.

Pizza Website

The Pizza Website was a final project for my introductory HTML and CSS class in 2021. I had to create a website entirely made out of HTML and CSS that met a certain set of criteria but the way of reaching that was left up to us. It consisted of 4 webpages, a Home, a Menu, a History of Pizza page, and a Customer Feedback page. The menu page has a long list of menu items and different pizza choices. The History of Pizza page has a history description and a large video player at the bottom that tells the history of pizza and a large yellow hand to pause/ play the video. This is a pretty low level bare-bones website, but I am very proud of it, because this is where my love for Web Dev began, creating a website that you created is a cool feeling. I don't plan to update this in the future, as it's a reminder to myself of where my passion began and how far I have come in being a developer.

The challenge I faced with website was getting the video to respond correctly to the play and stop functions of pressing the giant yellow hand next to the video player.
